# Keyboard shortcuts

| Shortcut | Action | Where |
|----------|--------|-------|
| `↩` (Return) | Primary action of the current screen | Welcome, Placement result, Answer feedback, Lesson summary, Home |
| `1`–`4` | Pick option N | Placement question, Lesson question |
| `⌘E` | Show examples for the current word | Answer feedback |
| `⌘,` | Open Settings | Anywhere |
| `Esc` | Close the active sheet (Examples panel, Settings) | Sheets |
| `Tab` / `Shift+Tab` | Move focus forward / backward | All screens |

Every interactive control above is also reachable via VoiceOver routing.
